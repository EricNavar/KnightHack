TESTING AND DEMO

Steps to Success

If you have already completed steps 1 and 2 then you are almost ready for your EPIC WIN. But before we get to that EPIC win we have to complete the final step. The final step includes testing out / debugging your code to make sure there are no errors and that you are ready to submit your submission to be ready for demoing.

  1. First be sure to go over all your code to make sure there are no errors or warnings that may cause errors.
  2. Be on the lookout also for syntax errors that can be easily missed.
  3. Make a test file that has code that will test against the code you wrote.
  4. Once you are sure that your code is complete with no errors and is working accordingly you are ready to submit your code for submission
  5. CONGRATULATIONS YOU HAVE WON THE HACKATHON EPIC WIN.
